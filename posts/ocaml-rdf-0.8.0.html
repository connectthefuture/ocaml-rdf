OCaml-RDF 0.8.0
January 21, 2014

Release 0.8.0 of OCaml-RDF is available from the download page. Changes:

  • Change Rdf_rdf.dc to http://purl.org/dc/elements/1.1/,
  • Add convenient functions to create RDF vocabularies in Rdf_rdfs,
  • Fix: handle more than one FROM <iri> directives in sparql query according to specification (merge referenced graphs in a new graph in memory).
