OCaml-RDF is available in OPAM:

$ opam install rdf

Compiling OCaml-RDF requires the following software to be installed:

  • OCaml >= 4.02,
  • Findlib >= 1.3.3 (should work with priori versions),
  • OCaml-iri >= 0.2.0,
  • OCaml-uri >= 1.9.1,
  • Calendar >= 2.03.2,
  • Xmlm >= 1.1.1,
  • Uuidm >= 0.9.5,
  • Sedlex >= 1.99.2,
  • Menhir >= 20120123 (not tested with prior versions),
  • Cryptokit >= 1.7 (not tested with prior versions),
  • PCRE-OCaml >= 7.0.2 (not tested with prior versions),
  • Yojson >= 1.1.8 (not tested with prior versions),
  • [optional] Lwt >= 2.4.5 and CoHTTP >= 0.11.2 (not tested with prior versions) to use the Lwt-based HTTP binding of the Sparql protocol,
  • [optional] OCaml-MySQL >= 1.1.1 (not tested with prior versions) for MySQL backend,
  • [optional] Postgresql-ocaml >= 2.0.2 (not tested with prior versions) for Postgresql backend,
  • [optional] Config-file >= 1.1 to run the test suite,

After extracting the source tree, go to the extracted directory and type:

$ ./configure $ make all $ make install # with sufficient rights

This will install the packages rdf, rdf.ppx, rdf.js, rdf.mysql,rdf.postgresql and rdf.lwt with ocamlfind.
