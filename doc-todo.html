Limitations and TODO
Limitations

The Sparql 1.1 query language is implemented, with some exceptions:

  • The following built-in calls are still missing: UCASE, LCASE, langMatches (only based on prefix by now, must implement language matching according to RFC 4647), TIMEZONE, TZ, SHA384, SHA512.
  • Subqueries are not handled yet.
  • DESCRIBE queries are not handled yet.
  • Check of variable scope is not implemented yet.
